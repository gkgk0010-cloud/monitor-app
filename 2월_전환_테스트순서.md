# 2월 전환 테스트 순서

테스트용 계정 정리 후, 회원가입부터 2월 유저프로필 생성·헤더 복사까지 확인하는 순서입니다.

---

## 1단계: 테스트 계정 정리

1. **스프레드시트** 열기 (똑패스용 시트)
2. **User_Profile_2026-02** 시트로 이동
3. 2월 전환 테스트로 썼던 **계정 행** 찾기 (이름/uid로 확인)
4. 해당 **행 전체 선택** → 우클릭 → **행 삭제** (또는 행 번호 우클릭 → 삭제)
5. (선택) **시트 자체를 삭제**하고 아래 회원가입으로 “시트가 없을 때 새로 만들기”까지 테스트하려면, **User_Profile_2026-02** 시트 탭 우클릭 → **삭제**  
   → 그러면 다음 회원가입 시 코드가 지난달(1월) 시트를 보고 2월 시트를 다시 만들게 됨

---

## 2단계: 1월 시트 헤더 확인 (기준용)

1. **User_Profile_2026-01** 시트 열기
2. **1행(헤더)** 확인: A열 uid, B열 name, C열 score, D열 streak, E열 last_solve_date, F열부터 **태그별 _Try, _Cor** 등이 있는지 확인
3. (기억용) 1행이 **몇 번째 열까지** 있는지, 또는 **마지막 열 글자**(예: AP) 정도만 메모해 두면, 나중에 2월과 비교하기 좋음

---

## 3단계: 회원가입

1. **똑패스** 웹앱 접속 (배포된 주소 또는 테스트용 주소)
2. **로그아웃** 또는 **시크릿/다른 브라우저**에서 진행
3. **회원가입** 진행: 이름, 반, 인증코드 등 입력 후 가입 완료
4. 가입 완료 메시지가 나오면 **3단계 완료**

---

## 4단계: 2월 유저프로필 시트 생성 여부 확인

1. 스프레드시트로 돌아가기
2. 시트 탭 목록에서 **User_Profile_2026-02** 가 있는지 확인
   - **1단계에서 2월 시트를 삭제했다면** → 이 단계에서 **새로 생겼는지** 확인
   - **이미 있었으면** → 그대로 두고 5단계로

---

## 5단계: 2월 시트 1행(헤더)이 1월과 똑같은지 확인

1. **User_Profile_2026-02** 시트 클릭
2. **1행** 확인:
   - A열 `uid`, B열 `name`, C열 `score`, D열 `streak`, E열 `last_solve_date`, F열 `상태` 등 **기본 열**이 1월과 같은지
   - **F열(또는 G열) 이후**에 1월처럼 **태그별 _Try, _Cor** 등이 **그대로 복사**되어 있는지
3. 1월 1행과 2월 1행을 **나란히 비교**해서 열 개수·이름이 같으면 **통과**

---

## 6단계: 새로 가입한 사용자 행 확인

1. **User_Profile_2026-02** 시트에서 **가장 아래 행** (또는 방금 가입한 이름이 있는 행) 확인
2. **A열**: 가입한 uid  
   **B열**: 가입한 이름  
   **C열 이후**: 점수·스트릭·날짜 등은 비어 있거나 기본값이어도 됨 (첫 가입이므로)
3. **F열 이후** 태그 열들은 **헤더만 있고 값은 비어 있는 상태**면 정상 (아직 문제를 안 풀었으므로)

---

## 7단계: (선택) 문제 풀기 → 태그 열 반영 확인

1. 방금 가입한 계정으로 **로그인** → **문제 풀기** 1~2문제 진행
2. 스프레드시트 **User_Profile_2026-02** 에서 해당 사용자 행 다시 확인
3. **해당 문제 태그**에 해당하는 **_Try**, **_Cor** 열에 숫자가 **1 이상**으로 들어갔는지 확인
4. 들어갔으면 **2월 전환 + 헤더 복사 + 데이터 반영**까지 모두 정상 동작한 것

---

## 체크리스트 요약

- [ ] 1단계: 테스트 계정 행 삭제 (또는 2월 시트 삭제)
- [ ] 2단계: 1월 시트 1행(헤더) 확인
- [ ] 3단계: 새 계정으로 회원가입
- [ ] 4단계: User_Profile_2026-02 시트 존재 여부 확인
- [ ] 5단계: 2월 1행이 1월과 똑같이 복사됐는지 확인
- [ ] 6단계: 새 사용자 행이 2월 시트에 추가됐는지 확인
- [ ] 7단계: (선택) 문제 풀어서 태그 열 반영 확인

이 순서대로 하시면 2월 전환과 “기존 행(헤더) 복사”가 제대로 되는지 끝까지 확인할 수 있습니다.
